Key Features

🌐

Network Scanning

Automatically discovers all active hosts and open ports on your local network using Nmap with service version detection.

🤖

AI Risk Scoring

Intelligently classifies each vulnerability as CRITICAL, HIGH, MEDIUM, or LOW based on CVE severity ratings and OWASP guidelines.

🔐

AES-256 Encryption

Protects all generated reports with military-grade AES-256 encryption and RSA-2048 key exchange — the same standard used by governments.

📊

Web Dashboard

Beautiful real-time dashboard in your browser showing color-coded vulnerability results with severity badges.

📧

Email Alerts

Automatically sends email notifications when CRITICAL or HIGH vulnerabilities are detected on your network.

🔥

Firewall Suggestions

Generates ready-to-run Windows Firewall commands to immediately block dangerous open ports with one click.

Installation Guide

1

Install Python 3.x

Download and install Python from the official website.

→ python.org/downloads
2

Install Nmap 7.99

Download Nmap and enable WinPcap API-compatible mode during installation.

→ nmap.org/download.html
3

Install Requirements

Open terminal in the VulnScanner folder and run:

pip install python-nmap pycryptodome reportlab flask
4

Run VulnScanner

Start the dashboard with a single command:

python dashboard/app.py

Then open your browser at localhost:5000
